实战工作流

工作流1: 竞品调研

# 1. 搜索竞品信息
web_search "竞品名称 + 功能介绍 + 2024"

# 2. 获取竞品官网内容
web_fetch "https://竞品官网.com"

# 3. 分析GitHub开源项目
github search repos --topic "竞品关键词" --sort stars

# 4. 监控社媒反馈
twitter-automation search "竞品名称" --sentiment negative

# 5. 生成调研报告
openclaw "基于以上信息，生成竞品分析报告"

工作流2: 趋势周报

# 1. 获取最新博客文章
blogwatcher fetch --category "AI" --since "1 week ago"

# 2. 搜索本周热点
web_search "AI行业动态 本周" --freshness pw

# 3. 总结关键内容
summarize --urls "文章1,文章2,文章3" --format bullet

# 4. 生成周报
openclaw "生成本周AI行业趋势周报"

工作流3: 论文速览

# 1. 下载论文PDF
web_fetch "https://arxiv.org/pdf/xxx.pdf"

# 2. 提取论文内容
pdf extract --file paper.pdf

# 3. 生成摘要
summarize --file paper.pdf --format academic

# 4. 保存到知识库
obsidian add "论文摘要" --vault Research

工作流4: 开源项目评估

# 1. 获取项目信息
github repo view owner/repo

# 2. 分析Issue反馈
gh-issues list --repo owner/repo --label bug

# 3. 查看代码趋势
github insights --repo owner/repo --metric commits

# 4. 生成评估报告
openclaw "评估这个开源项目的成熟度和采用价值"
