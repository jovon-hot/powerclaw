🔧 故障排除

问题 1: 响应慢

解决: 使用更简单的提示，明确指令

# 不好的示例

openclaw exec "帮我分析一下"



好的示例


openclaw exec "分析这个 CSV 的前 10 行数据，找出异常值"

问题 2: 结果不准确

解决: 提供更多上下文

# 添加背景信息

openclaw exec "基于我们之前的讨论，继续优化这个方案..."

问题 3: 格式化问题

解决: 明确指定输出格式

openclaw exec "用 Markdown 表格格式输出结果"

openclaw exec "用 JSON 格式返回数据"
