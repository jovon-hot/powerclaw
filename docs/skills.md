# 精品 Skill 推荐

OpenClaw 的强大之处在于丰富的技能生态系统。以下是经过验证的精品技能推荐。

---

## 🌟 必装技能

### 1. **web-search** - 网络搜索增强
**功能**: 让 AI 能够实时搜索网络信息

**使用场景**:
- 获取最新信息
- 验证事实
- 研究竞品
- 监控趋势

**安装**:
```bash
openclaw skills install web-search
```

**示例**:
```bash
openclaw exec "搜索 OpenAI 最新的产品发布"
openclaw exec "查找 2025 年最好的 DeFi 项目"
```

## 🚀 快速开始

### 安装多个技能
```bash
# 安装所有推荐技能
openclaw skills install web-search browser file-system messaging git
```

### 查看已安装技能
```bash
openclaw skills list
```

### 更新技能
```bash
openclaw skills update web-search
```

### 卸载技能
```bash
openclaw skills uninstall old-skill
```

## 🎓 技能开发

想要开发自己的技能？

### 技能结构
```
my-skill/
├── skill.yaml      # 技能配置
├── index.js        # 主入口
└── README.md       # 说明文档
```

### 基本模板
```javascript
// index.js
module.exports = {
  name: 'my-skill',
  version: '1.0.0',
  
  async execute(context, params) {
    // 实现技能逻辑
    return result;
  }
};
```

### 发布技能
```bash
openclaw skills publish ./my-skill
```
