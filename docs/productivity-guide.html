实战工作流组合

工作流1: 每日工作计划

# 1. 查看今日提醒
apple-reminders list --list 今天

# 2. 获取日历事项
gog calendar today

# 3. 检查GitHub待办
gh-issues list --assignee @me

# 4. 生成今日计划
openclaw "根据以上信息，帮我制定今日工作计划"

工作流2: 智能会议助手

# 1. 录音转文字
openai-whisper record --output meeting.mp3

# 2. 生成会议纪要
summarize --file meeting.mp3 --format markdown

# 3. 提取行动项
openclaw "从会议纪中提取行动项，并添加到Things"

# 4. 创建任务
things-mac add "完成会议跟进" --list 工作

工作流3: 开源项目维护

# 1. 获取待处理Issue
gh-issues list --label "bug" --sort updated

# 2. 自动修复（如果适用）
code-review --fix-suggestions

# 3. 生成发布说明
release-manager changelog

# 4. 发布新版本
gh release create v1.2.0
