精品 Skill 推荐

OpenClaw 的强大之处在于丰富的技能生态系统。以下是经过验证的精品技能推荐。

🌟 必装技能

1. web-search - 网络搜索增强

功能: 让 AI 能够实时搜索网络信息

使用场景:

  • 获取最新信息
  • 验证事实
  • 研究竞品
  • 监控趋势

安装:

openclaw skills install web-search

示例:

openclaw exec "搜索 OpenAI 最新的产品发布"

openclaw exec "查找 2025 年最好的 DeFi 项目"

2. browser - 浏览器自动化

功能: 控制浏览器，自动化网页操作

使用场景:

  • 自动填写表单
  • 网页数据抓取
  • 自动化测试
  • 截图监控

安装:

openclaw skills install browser

示例:

openclaw exec "打开 GitHub 并搜索 'openclaw'"

openclaw exec "截取 Google 首页的截图"

3. file-system - 文件系统操作

功能: 增强的文件读写和管理能力

使用场景:

  • 批量文件处理
  • 自动备份
  • 文件转换
  • 目录管理

安装:

openclaw skills install file-system

示例:

openclaw exec "将当前目录的所有 .txt 文件转换为 .md"

openclaw exec "备份 ~/Documents 到 ~/Backups"

4. messaging - 消息发送

功能: 发送邮件、Slack、Telegram 消息

使用场景:

  • 定时通知
  • 报告发送
  • 告警提醒
  • 团队协作

安装:

openclaw skills install messaging

示例:

openclaw exec "发送邮件给 team@example.com，主题是今日报告"

openclaw exec "发送 Slack 消息到 #general 频道"

🚀 快速开始

安装多个技能

# 安装所有推荐技能

openclaw skills install web-search browser file-system messaging git

查看已安装技能

openclaw skills list

更新技能

openclaw skills update web-search

卸载技能

openclaw skills uninstall old-skill

🎓 技能开发

想要开发自己的技能？

技能结构

my-skill/

├── skill.yaml      # 技能配置 ├── index.js        # 主入口 └── README.md       # 说明文档

基本模板

// index.js

module.exports = {   name: 'my-skill',   version: '1.0.0',



  async execute(context, params) {     // 实现技能逻辑     return result;   } };

发布技能

openclaw skills publish ./my-skill
